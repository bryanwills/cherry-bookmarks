## Development

```bash
# install dependencies
pnpm i

# create a .env file
cp env.example .env

# run the app in dev mode
pnpm dev

# you may want to create an user
pnpm cherry create-user "foo@example.com" "P@ssw0rd"
pnpm cherry set-admin "foo@example.com"
```
